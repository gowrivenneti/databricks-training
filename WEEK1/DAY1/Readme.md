# SQL Practice Queries Collection

A comprehensive collection of SQL practice queries designed to strengthen fundamental and intermediate SQL concepts using a simple Employee–Department–Project database schema.



# 📌 Project Overview

This project contains **65 SQL practice queries** covering a wide range of database operations including:

- Data Retrieval
- Filtering
- String Matching
- Date Operations
- Aggregate Functions
- GROUP BY & HAVING
- Sorting
- Joins
- Subqueries
- Correlated Queries
- Moderate-Level SQL Problems

This repository is ideal for:

- Students learning SQL
- Placement preparation
- SQL interview practice
- DBMS lab practice
- Beginners in databases



# 🗂 Database Schema

## Employee Table

| Column Name | Description |
|-------------|-------------|
| emp_id | Unique employee ID |
| name | Employee name |
| age | Employee age |
| salary | Employee salary |
| department_id | Department reference ID |
| hire_date | Employee hiring date |

---

## Department Table

| Column Name | Description |
|-------------|-------------|
| department_id | Unique department ID |
| name | Department name |

---

## Project Table

| Column Name | Description |
|-------------|-------------|
| project_id | Unique project ID |
| name | Project name |
| department_id | Associated department ID |
